1. Install Python Dependencies

# Using virtual environment
# Windows:
python -m venv venv
# macOS:
python3 -m venv venv

# Activate virtual environment
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# Install packages
# Windows:
pip install -r requirements.txt
# macOS:
pip3 install -r requirements.txt
